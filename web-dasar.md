# Praktikum HTML Dasar

## Tujuan
Mahasiswa memahami struktur dasar HTML.

## Persiapan
- Browser
- Text Editor

## Langkah 1: Buat File HTML
Buat file `index.html`

```html
<!DOCTYPE html>
<html>
<head>
  <title>Hello</title>
</head>
<body>
  <h1>Hello Mahasiswa</h1>
</body>
</html>
