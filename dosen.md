# Dosen
## Wiga Mobile
Wiga Mobile adalah ekosistem digital yang akan terus dikembangkan dilingkungan institut teknologi dan bisnis widya gama lumajang sebagai media digital dalam merekam proses belajar mengajar hingga administrasi lainnya.
## Login
untuk memulai menggunakan aplikasi wiga mobile yang harus diperhatakan adalah akses kedalam ekosistem tersebut.
berikut cara untuk login pada aplikasi wiga mobile
- akses url : https://sso.itbwigalumajang.ac.id
- masukkan username : Nomor Induk Mahasiswa (NIM)
- masukkan password : Personal Identification Number (PIN)
- klik login.
